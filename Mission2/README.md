# VertexMind Internship - Mission 2

## Project Name

Student Management System

## Overview

This project demonstrates Object-Oriented Programming (OOP) concepts in Python using classes, inheritance, lists, dictionaries, loops, and error handling.

## Features

* Parent and Child Classes
* Inheritance using `super()`
* Student data storage
* Dictionary for marks management
* List for storing multiple students
* Error handling using try/except
* Console-based output

## Technologies Used

* Python 3
* VS Code
* GitHub

## Concepts Learned

* Classes and Objects
* Constructors (`__init__`)
* Inheritance
* Lists and Dictionaries
* Loops
* Exception Handling
