# Student Task Manager

## Overview

Student Task Manager is a simple Command Line Interface (CLI) application developed using Python.
This project was created as part of the VertexMind Python Internship program to practice real-world Python development concepts.

The application helps users organize and manage daily tasks directly from the terminal.

---

# Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Remove tasks
* User-friendly CLI interface
* Colorful terminal output using Colorama
* Error handling using try/except
* Unit testing using pytest

---

# Installation

Clone the repository:

```bash
git clone https://github.com/Anushka04122/VertexMind_Python_Mission1.git
```

Move into the project folder:

```bash
cd VertexMind_Python_Mission1/Mission3
```

Install required dependencies:

```bash
pip install -r requirements.txt
```

---

# Running the Project

Run the application using:

```bash
python main.py
```

---

# Running Tests

This project uses pytest for unit testing.

Run tests using:

```bash
pytest
```
